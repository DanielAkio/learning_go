# Bookings and Reservations

This is the repository for my bookings and reservations project.

## Prerequisites

- Golang
 - buffalo soda
- Docker

## Migrations (soda)
- ```soda generate fizz <migration_name>```
- ```soda migrate```
- ```soda migrate down -s <number_of_migrations>```

## Running Local

- Git Bash
  - `./run.sh`
- Powershell
  - `./run.bat`
